# Timezone Tray Clock 🕒

一个轻量、优雅且达到生产级（Production-Ready）标准的 Windows 桌面时钟微件。基于 **.NET 8.0** 与 **WPF** 构建，旨在为你提供一个悬浮于桌面右下角、免打扰且高度可定制的异地时间监控工具。

## ✨ 核心特性

- **悬浮与穿透**：默认置顶显示（甚至能悬浮在 Windows 任务栏上方）。支持**锁定位置（鼠标穿透）**，绝不干扰你的日常办公和游戏。
- **无极缩放**：把鼠标悬停在时钟上，按住 `Ctrl + 鼠标滚轮` 即可无极调节时钟大小（支持 50% ~ 300% 缩放），满足各种视力和屏幕尺寸需求。
- **动态主题适配**：内置优雅的玻璃质感 UI。支持**自动跟随系统**切换深色/浅色模式，并配有对应的动态外阴影效果。
- **一键隐藏/显示**：双击系统右下角的托盘图标即可瞬间隐藏时钟；隐藏后自动暂停后台计算，实现 **零 CPU 占用休眠**。
- **生产级稳定性**：
  - **防丢机制**：完美适配多显示器环境。当拔除副屏导致坐标越界时，时钟会自动安全回退到主屏幕，告别“时钟走丢”。
  - **防崩溃原子保存**：用户的偏好设置采用 `.tmp` 文件加 NTFS 原子覆盖机制，即使在修改设置的瞬间意外断电或蓝屏，配置也不会损坏。
  - **防关机阻拦**：底层采用安全的物理键盘事件拦截，完美防范 `Alt+F4` 误杀，且绝不会阻拦 Windows 系统的正常注销与关机。

## 🚀 快速上手

1. **运行**：双击 `timezone-tray-clock.exe` 即可运行，默认会自动停靠在屏幕右下角。
2. **移动位置**：用鼠标左键按住时钟即可随意拖拽。
3. **设置中心**：右键点击 Windows 任务栏右下角的 🕒 托盘图标，可以进行所有设置：
   - 切换主题颜色（跟随系统 / 始终深色 / 始终浅色）
   - 开启/关闭鼠标穿透
   - 设置开机自启
   - 调整预设缩放比例
4. **快捷键**：鼠标悬停在时钟上时，使用 `Ctrl + 滚轮` 自由放大和缩小。

## 🛠️ 技术栈与编译

- **框架**：.NET 8.0 Windows Desktop (WPF)
- **托盘支持**：[H.NotifyIcon.Wpf](https://github.com/HavenDV/H.NotifyIcon) (v2.4.1)

### 本地编译

确保你的电脑上安装了 [.NET 8.0 SDK](https://dotnet.microsoft.com/download/dotnet/8.0)。然后在项目根目录执行：

```bash
# 编译并运行
dotnet run

# 构建独立的单文件发布版本 (Self-contained)
dotnet publish -c Release -r win-x64 --self-contained true -p:PublishSingleFile=true
```
