# Timezone Tray Clock 🕒

一个轻量、优雅且达到生产级（Production-Ready）标准的 Windows 桌面时钟微件。基于 **.NET 8.0** 与 **WPF** 构建，旨在为你提供一个悬浮于桌面右下角、免打扰且高度可定制的异地时间监控工具。无论你是进行跨国协作还是关注外汇交易时间，这都是你桌面上最完美的时钟伴侣。

---

## 📦 如何编译与打包发布

为了让你能直接将这个应用发给朋友或者部署到任何一台 Windows 电脑上（无需他们额外安装 .NET 环境），你可以将其打包为独立的单文件应用。

**前提要求：**
确保你的电脑上安装了 [.NET 8.0 SDK](https://dotnet.microsoft.com/download/dotnet/8.0)。

**执行打包：**
打开命令行终端，定位到项目根目录，然后执行以下命令：

```bash
# 编译并运行 (用于开发测试)
dotnet run

# 构建独立的单文件发布版本 (Self-contained)
dotnet publish -c Release -r win-x64 --self-contained true -p:PublishSingleFile=true
```

打包完成后，在 `bin/Release/net8.0-windows/win-x64/publish/` 目录下会生成一个单独的 `timezone-tray-clock.exe`。你只需要把这个 EXE 文件拿走，双击即可运行！

## 🎮 如何使用

一旦你运行了本程序，时钟卡片会自动停靠在你的屏幕右下角。

1. **基本移动**：鼠标左键按住时钟即可在屏幕上随意拖拽。
2. **快捷缩放 (推荐)**：把鼠标悬停在时钟上，按住键盘的 `Ctrl` 键，同时**上下滚动鼠标滚轮**，即可体验丝滑的无极缩放效果（从迷你到超大）。
3. **高级设置 (托盘菜单)**：在 Windows 任务栏右下角的 🕒 托盘图标上点击**右键**，可以呼出设置菜单。在这里你可以：
   - 切换主题（支持跟随系统、强制浅色或深色）
   - 调整预设的缩放比例（85% ~ 150%）
   - **锁定位置（开启鼠标穿透）**：开启后，鼠标点击会直接穿透时钟点到背后的窗口，再也不怕办公/打游戏时误触。
   - 开启/关闭开机自启
4. **一键隐藏/显示**：如果你暂时不需要看时钟，可以**直接双击**系统任务栏右下角的 🕒 托盘图标，时钟会瞬间隐藏（且暂停后台计算），再次双击即可唤醒。

## ✨ 核心亮点与架构设计

本项目不仅是一个外观精美的微件，其底层更是一个应对各种异常场景的现代化 WPF 参考工程：

- **零 CPU 占用休眠**：隐藏时钟时，底层的 `DispatcherTimer` 将被物理挂起，释放全部计算资源。
- **坚不可摧的持久化层**：采用 Windows `NTFS` 原子级写入机制 (`.tmp` + Replace) 进行 JSON 配置的保存。就算在修改设置的瞬间意外断电或死机，你的偏好配置也不会产生丝毫损坏。
- **动态防丢坐标校正**：拥有完善的虚拟屏幕 (`VirtualScreen`) 边界判定。如果你将时钟拖入副屏，而在拔掉副屏后重启程序，时钟会自动瞬移回主屏幕安全区，彻底杜绝“屏幕外幽灵窗口”问题。
- **无副作用防误杀**：全面接管 `OnKeyDown` 级别的按键拦截，防止用户随手按下 `Alt + F4` 误杀程序，同时也保证了 Windows 操作系统的正常关机流程不受任何阻碍。
- **始终位于任务栏之上**：通过高频刷新的 `EnsureTopmost` 调用，即使面对同样拥有最顶层权限的 Windows 任务栏，时钟依然能保证自己不会被盖住。
