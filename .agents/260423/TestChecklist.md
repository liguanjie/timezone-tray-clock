# Timezone Tray Clock - 手动验证测试清单

为了验证重构后的“钢铁之躯”，请按照以下步骤逐一测试程序：

## 1. 核心交互与体验验证 🖱️

*   [ ] **托盘双击显隐**：双击系统任务栏右下角的时钟托盘图标。
    *   *预期结果*：时钟立即消失；再次双击，时钟重新出现并在屏幕最顶层。
*   [ ] **悬浮提示 (ToolTip)**：将鼠标移动到时钟卡片上方并停留 1 秒。
    *   *预期结果*：浮现出文本提示（例如 `China Standard Time`）。
*   [ ] **主题阴影随动**：右键托盘 -> 主题颜色，在“始终深色”和“始终浅色”之间切换。
    *   *预期结果*：不仅背景色和文字颜色改变，时钟卡片**背后的边缘阴影**也会根据深浅色模式自动微调，视觉更加立体。
*   [ ] **高清 ICO 图标**：查看托盘右下角的图标。
    *   *预期结果*：显示为我们生成的清晰应用图标，并且应用能够正常启动（不再抛出之前的 `ArgumentException` 崩溃）。

## 2. 极端防御场景验证 🛡️

*   [ ] **防误杀 (Alt+F4 拦截)**：用鼠标点击一下时钟卡片（确保它没有处于鼠标穿透模式），使其获得焦点，然后按下键盘的 `Alt + F4`。
    *   *预期结果*：什么都不会发生。时钟不会关闭，坚如磐石。
*   [ ] **多屏幕防丢测试 (模拟越界)**：
    1. 右键托盘 -> 退出程序。
    2. 按 `Win + R` 输入 `%APPDATA%\TimezoneTrayClock` 打开文件夹，用记事本打开 `settings.json`。
    3. 把 `WindowLeft` 改为 `9999`，`WindowTop` 改为 `9999`（模拟副屏被拔掉后的残留坐标），保存文件。
    4. 重新启动程序。
    *   *预期结果*：程序检测到 9999 的坐标已经飞出屏幕边界，会自动触发安全回退，把时钟重新稳稳地放在你当前屏幕的右下角。
*   [ ] **注册表幽灵路径防御**：
    1. 右键托盘 -> 勾选“开机自启动”。
    2. 按 `Win + R` 输入 `regedit`，定位到 `HKEY_CURRENT_USER\Software\Microsoft\Windows\CurrentVersion\Run`。
    *   *预期结果*：你会在右侧看到 `TimezoneTrayClock`，其数据值是你当前启动 `.exe` 的**绝对物理路径**，不再是空路径。

## 3. 数据安全验证 💾

*   [ ] **配置静默保存修复**：
    1. 退出程序，删除 `%APPDATA%\TimezoneTrayClock` 文件夹（模拟全新用户第一次安装）。
    2. 启动程序，**此时不要拖动时钟**，直接右键托盘勾选“锁定位置(鼠标穿透)”。
    3. 退出程序。
    *   *预期结果*：再次打开 `%APPDATA%\TimezoneTrayClock`，你会发现 `settings.json` 被成功创建，且 `IsClickThroughEnabled` 被正确保存为了 `true`。（旧代码由于 `NaN` 的阻碍，这一步会静默失败）。
